## About

SherrodBots is an AI consulting website for Sherrod Seward, helping businesses deploy custom AI agents including:

- **AI Voice Agents** - Handle calls in 10+ languages
- **Email Assistants** - Draft emails in your exact voice
- **Document Intelligence** - Process thousands of docs automatically
- **Chat & SMS Bots** - 24/7 automated conversations
- **Workflow Automation** - Connect your tools seamlessly

## Tech Stack

- **Next.js 16** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Stripe** for payments (placeholder)
- **Sevyn** AI chat widget

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Create a `.env.local` file:

```
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_PLACEHOLDER
STRIPE_SECRET_KEY=sk_live_PLACEHOLDER
```

Deployed on Netlify with automatic builds from GitHub.
